# Unity Client

This folder is reserved for the Unity desktop client used to render character panels.

## Responsibilities

- Load character models with lip sync
- Play animations aligned with audio segments
- Communicate with backend via REST or WebSocket
- Send interrupt events for live Q&A

## Setup

1. Open the project in Unity (2022 LTS recommended)
2. Configure the backend URL in the client settings
3. Import character assets and animations
4. Play the sample scene to verify audio sync
